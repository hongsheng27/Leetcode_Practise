0435. Non-overlapping Intervals
題目概述
給一堆 interval，移除最少的 interval 使得剩下的全部不重疊。回傳最少需要移除幾個。
例：[[1,2],[2,3],[3,4],[1,3]]，移除 [1,3] 後剩下三個不重疊，答案是 1。
注意：端點相接（如 [1,2] 和 [2,3]）不算重疊。

解題邏輯
等價轉換：移除最少 = 保留最多不重疊的 interval。
Greedy 核心：按 end 排序，優先保留 end 最早的 interval——它結束得早，給後面的 interval 留最多空間（等同排課表時優先選最早結束的課）。
遍歷時追蹤 preEnd（目前保留的最後一個 interval 的 end）：

start >= preEnd → 不重疊，保留，更新 preEnd
start < preEnd → 重疊，移除，res += 1（不更新 preEnd，排序已保證當前 end ≥ preEnd，保留較小的更好）


Time O(n log n)，排序主導。Space O(1)。
