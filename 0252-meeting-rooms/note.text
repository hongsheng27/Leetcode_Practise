
LeetCode 252 - Meeting Rooms
==============================

題目：給一組會議時間 intervals，判斷一個人能不能參加所有會議（有沒有時間重疊）。

Interval 概念：
  每個 interval 是一段時間 [start, end]。
  兩個 interval 重疊 = 一個還沒結束，另一個就開始了。

解題思路：
  暴力解：每一對都互相比較有沒有重疊 → O(n^2)。
  優化：按 start 排序後，只需要比較相鄰的兩個 → O(n log n)。
  因為排序後如果 A 和 C 重疊，A 和 B 一定也重疊（B 夾在中間）。
  所以只要相鄰沒重疊，全部就都沒重疊。

  判斷條件：下一個的 start < 前一個的 end → 重疊 → return False。

踩過的坑：
  1. 必須先理解暴力解需要比對所有配對有沒有 overlap。
     排序的意義是把問題簡化成只看相鄰的，一有重疊即可回傳 False。
     不理解這一步的話，會不知道為什麼排序就夠了。

複雜度：O(n log n) 時間（排序），O(1) 空間。
