LeetCode 853 - Car Fleet
=========================

題目：一條單線道，多台車從不同位置以不同速度開往終點 target。
後車追上前車時會合併成車隊（不能超車），速度變成前車的速度。
問最後有幾個車隊到達終點。

關鍵觀察：
  先算每台車到終點所需時間：time = (target - position) / speed
  按位置排序後，從最靠近終點的車開始往回看：
    - 後車 time <= 前車 time → 後車追得上，被吸收進前車的車隊，跳過
    - 後車 time > 前車 time → 追不上，形成新的車隊，push 進 stack
  stack 裡留下的就是每個獨立車隊，len(stack) 就是答案。

本質上是 monotonic stack：stack 從底到頂維持遞增的 time。
比前車快的（time 更小的）都會被吸收掉，只有更慢的才留下來。


複雜度：O(n log n) 時間（排序），O(n) 空間。
