703. Kth Largest Element in a Stream
     ====================================

題目概述：
設計一個資料結構，持續加入數字。
每次 add(val) 後，回傳目前資料中的第 k 大元素。

解題邏輯：
使用 size = k 的 min heap。

heap 裡只保留目前最大的 k 個數字。
如果 heap size 超過 k，就 pop 掉最小值。

因此：

```
heap[0]
```

是「最大 k 個數字中最小的」，也就是第 k 大。

踩過的坑：
無。這題有直接想到使用 size = k 的 min heap。

複雜度：
初始化：
O(n log k)
add：
O(log k)
空間複雜度：
O(k)
