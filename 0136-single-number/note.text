136. Single Number

題目概述： 給一個整數陣列，每個數字都出現兩次，只有一個數字出現一次，找出那個落單的數字。要求 O(1) space。
解題邏輯： 把所有數字 XOR 在一起。因為 a ^ a = 0、a ^ 0 = a，成對的數字互相抵消變 0，最後剩下的就是答案。
踩過的坑：

看到 O(1) space 先想工具：two pointer 無法處理無序多數字的情況，XOR 的自反性（成對消除）剛好匹配「出現兩次 / 找唯一」的題目特徵。
XOR 記法：不同才是 1（exclusive OR = 排他的 OR）。每個 bit 獨立比較，不同給 1，一樣給 0。

觸發關鍵字： 看到「成對 / 出現兩次 / 找唯一一個」→ XOR。
複雜度： O(n) time / O(1) space
