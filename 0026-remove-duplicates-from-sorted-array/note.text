26. Remove Duplicates from Sorted Array

題目概述： 給一個已排序的整數陣列 nums，in-place 把重複元素移到後面，讓前 k 個位置是不重複的元素且保持原順序，回傳 k。
解題邏輯： 快慢指標。s 是已確認結果的邊界，f 用 for 迴圈掃描每個元素。nums[f] != nums[s] 時，s += 1 再把 nums[f] 寫入 nums[s]，代表收進結果。相同就跳過。最後回傳 s + 1。

踩過的坑：
一開始糾結 s 和 f 之間的互動關係——「相等時 s 要幹嘛、不等時 f 要幹嘛、if 成立後 s/f 怎麼動」，越想越混亂。
正確思維是：f 永遠在跑，每個元素只問一個問題——「這個值跟結果最後一個一不一樣」，不一樣 → s 前進一格，把 nums[f] 寫進去
一樣 → 什麼都不做，f 自己往下走。決策主體是 nums[f]，不是兩個指標的互動。


複雜度： O(n) time / O(1) space
