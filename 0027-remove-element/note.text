27. Remove Element
題目概述： 給一個整數陣列 nums 和一個值 val，in-place 移除所有等於 val 的元素，回傳剩餘元素的長度 k。
解題邏輯： 快慢指標模板。s 是寫入位置，f 用 for 掃描。nums[f] != val 就寫到 nums[s]，s += 1。等於 val 就跳過。回傳 s。
踩過的坑： 無，跟 Move Zeroes 和 Remove Duplicates 同模板，只換 if 條件。
模板筆記： 快慢指標去除類——s = 下一個該放東西的位置，f = 掃描員。合格就寫入 s 並前進，不合格就跳過。只換合格條件。
複雜度： O(n) time / O(1) space


雙指針設計原則：
兩個指針各自要有獨立的起點和方向。
如果其中一個需要「從另一個的位置重新出發」，通常是個警訊。
觸發關鍵字：
看到 in-place 移動/刪除/分類 → two pointer。
跟 prefix sum（subarray sum、區間查詢）完全不同。
