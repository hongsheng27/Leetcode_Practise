56. Merge Intervals

題目概述： 給一組區間 [[1,3],[2,6],[8,10],[15,18]]，如果任兩個區間有重疊（如 [1,3] 和 [2,6]），就合併成一個（[1,6]）。回傳所有合併後的不重疊區間 → [[1,6],[8,10],[15,18]]

解題邏輯： 先按 start 排序，把第一個區間放進 merged。從第二個開始，每個區間跟 merged[-1] 比較：如果 intervals[i][0] <= merged[-1][1] 代表重疊，更新 merged[-1][1] = max(merged[-1][1], intervals[i][1])；否則直接 append。

踩過的坑：
一開始拿 intervals[i] 跟 iｓntervals[i+1] 比較，忽略了 merge 會延長區間的結尾。例如 [[1,5],[2,3],[4,8]]，merge 後 [1,5] 跟 [4,8] 其實重疊，但如果拿原始的 intervals[1][1]=3 去比 intervals[2][0]=4 就會漏掉。
核心原則：結果會被修改時，永遠跟結果的最新狀態比（merged[-1]），不是跟原始輸入比。

複雜度： O(n log n) time / O(n) space
