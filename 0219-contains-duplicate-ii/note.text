
LeetCode 219 - Contains Duplicate II
======================================

題目：給一個陣列 nums 和整數 k，判斷是否存在兩個不同的 index i, j，
使得 nums[i] == nums[j] 且 abs(i - j) <= k。

Fixed Sliding Window 經典題。

核心思路：
  維護一個大小最多為 k 的 HashSet 當作窗口。
  從左往右掃，每次檢查新元素是否已經在窗口裡（代表 k 範圍內有重複）。
  窗口超過 k 時，把最左邊的元素移出。

  HashSet 的作用：O(1) 檢查「窗口內有沒有重複」。
  固定窗口大小 k：保證找到的重複一定在 k 的距離內。


複雜度：O(n) 時間，O(k) 空間。
