1046. Last Stone Weight
      ========================

題目概述：
給一組 stones。
每次拿出最重和第二重的石頭互撞。

如果重量相同：
兩顆都消失。

如果重量不同：
留下兩者重量差。

重複直到剩下最多一顆石頭，回傳最後重量。
如果沒有剩下石頭，回傳 0。

解題邏輯：
使用 max heap。

Python heapq 預設是 min heap，
所以把石頭重量轉成負數，模擬 max heap。

每一輪：
1. pop 最重的石頭
2. pop 第二重的石頭
3. 計算重量差
4. 如果重量差不為 0，再 push 回 heap

最後：
heap 有值 → 回傳剩下石頭重量
heap 為空 → 回傳 0

踩過的坑：
無。這題有直接想到使用 max heap 模擬。

複雜度：
時間複雜度：O(n log n)
heapify 是 O(n)。
每輪最多做固定次數的 pop / push，每次是 O(log n)。
最多執行 O(n) 輪。

空間複雜度：O(n)
heap 需要保存 stones。
